# Miao-Yunzai ## 环境准备 ## 1.安装nodejs 点击[https://nodejs.org/](https://nodejs.org/) 下载 一路next即可 按 Win+R 进入 cmd 查看是否安装成功，出现版本号，即为安装成功 ````md node -v #查看nodejs版本 npm -v #查看npm版本 ```` **再装一下pnpm** win+R打开cmd ````md npm install pnpm -g ```` ````md pnpm -v ```` 查看一下是否安装成功 ## 2.安装redis 点击[https://github.com/tporadowski/redis/releases](https://github.com/tporadowski/redis/releases) 下载 还是一路next ## 3.安装Git 点击[https://git-scm.com/download/win](https://git-scm.com/download/win)一路nex ## 环境配置完成，开启装崽 任意位置鼠标右键或者cmd，克隆喵崽 使用github ````md git clone --depth=1 https://github.com/yoimiya-kokomi/Miao-Yunzai.git cd Miao-Yunzai git clone --depth=1 https://github.com/yoimiya-kokomi/miao-plugin.git ./plugins/miao-plugin/ ```` 使用gitee ````md git clone --depth=1 https://gitee.com/yoimiya-kokomi/Miao-Yunzai.git cd Miao-Yunzai git clone --depth=1 https://gitee.com/yoimiya-kokomi/miao-plugin.git ./plugins/miao-plugin/ ```` 安装依赖 ````md pnpm install -P ```` ## 开始登陆 ````md node app ```` 协议包括签名api自己想办法，或者[Qsign](https://qsign.icu/)，配置完成就可以开心的跑崽了 托管 ````md npm run start ````
